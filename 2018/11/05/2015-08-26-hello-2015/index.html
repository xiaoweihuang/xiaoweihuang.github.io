Hello World

发表于 2018-11-05

“Hey It works! ”

今天终于把博客搭建好了。

作为一个喜欢折腾的人，总也喜欢尝试一些新鲜的东西。虽然不知道自己会不会经常来这里写东西，但还是把博客搭建起来，给自己一个涨姿势的机会。很久之前就开始关注使用GitHub Pages + Jekyll 搭建Blog，觉得挺有意思的，于是开始在网上搜集资料，结合官方文档介绍，在本地把环境搭建起来，到今天算是把整体框架搞定了。

“Thanks!”

就在一切准备就绪的时候，一执行”jekyll serve”命令,就报错”cannot load such file – hitimes/hitimes (LoadError)”。

通过谷歌找到了 解决方案 感谢Stack Overflow社区的答主Azolo 。
